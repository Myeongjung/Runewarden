## 📄 Full Description (Steam HTML 포맷)

```
[h1]당신의 덱이 곧 요새입니다.[/h1]

마법 카드를 구성하고, 타워를 소환하며, 쉬지 않고 밀려오는 적의 물결로부터 넥서스를 지키세요.
[b]Runewarden[/b]은 타워 디펜스 × 덱빌딩 × 로그라이크를 하나로 엮은 전략 하이브리드 게임입니다.

[img]{STEAM_CLAN_IMAGE}/screenshot_combat.png[/img]

[h2]핵심 게임플레이[/h2]

[b]카드 하나가 곧 전술이다[/b]
모든 카드는 실질적인 행동입니다.
[list]
[*] [b]소환 카드[/b] — 빈 헥스에 타워 배치 (Archer, Cannon, Frost Mage, Fire Drake, Tesla Coil, Druid Grove)
[*] [b]강화 카드[/b] — 기존 타워에 능력치 부스트 부착 (타워당 최대 2개)
[*] [b]주문 카드[/b] — 즉시 발동 마법 (빙결, 화염, 번개, 시간 왜곡...)
[/list]

[b]로그라이크 런 구조[/b]
매 런마다 다른 덱, 다른 선택, 다른 결과.
[list]
[*] 웨이브 클리어 후 상점·이벤트·휴식 노드 선택
[*] 카드 구매, 제거, 희귀 유물 획득
[*] 10종 랜덤 이벤트 (황금 유물, 저주 계약, 적 약화...)
[/list]

[b]덱빌딩 깊이[/b]
[list]
[*] 30종 카드 (Common / Uncommon / Rare)
[*] 상점 리롤, 카드 제거로 덱 최적화
[*] 웨이브 중 카드 플레이 가능 (골드 +1 추가 비용)
[/list]

[img]{STEAM_CLAN_IMAGE}/screenshot_shop.png[/img]

[h2]콘텐츠[/h2]
[list]
[*] [b]6종 타워[/b]: Archer · Cannon · Frost Mage · Fire Drake (번 DoT) · Tesla Coil (체인 번개) · Druid Grove (슬로우 + 인접 버프)
[*] [b]30종 카드[/b]: 소환 6 · 강화 10 · 주문 14
[*] [b]5웨이브 Act 1[/b] — Wave 5: Ironclad 보스 (HP 700, 황금 글로우)
[*] [b]5종 적[/b]: Grunt · Runner · Tank · Elite · Ironclad Boss
[*] [b]메타 진행[/b]: Warden Rank 1~20, Codex 언락 9종, 영구 보너스
[*] [b]Steam 업적[/b] 20개
[/list]

[h2]게임의 차별점[/h2]

[b]"카드를 플레이하는 것 = 타워 액션"[/b]
배치·업그레이드·스킬이 하나의 카드 자원으로 통합됩니다. 
Balatro의 카드 재미 + Kingdom Rush의 타워 배치 + Slay the Spire의 덱 최적화.

[*] 볼륨/뮤트 컨트롤 내장
[/list]
```
